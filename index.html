What is this?

Remastered Kali NetHunter app with Material You Design

Features

  • Fresh and nice design (Monet included)
  • Full compatibility with new versions of Android
  • Improved Chroot Manager (there is no chance to lose personal data from the sdcard partition, smarter installation, the ability to install chroot in a directory you specify, etc.)
  • Integration with Termux and NetHunter Terminal
  • In the future - the implementation of hidden attacks, the ability to modify the system for attacks
  • We accept your ideas for implementation here
